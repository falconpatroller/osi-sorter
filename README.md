# OSI Model Brick Sorter

Static HTML, CSS and JavaScript sorting activity for embedding in Canvas via an iframe.

## Files
- `index.html`
- `styles.css`
- `script.js`
- `assets/bulb-off.svg`
- `assets/bulb-green.svg`
- `assets/bulb-red.svg`

## GitHub Pages
Upload the whole folder to a GitHub repository and enable GitHub Pages.

## Canvas iframe example
```html
<iframe
  src="https://YOUR-USERNAME.github.io/YOUR-REPO/"
  width="100%"
  height="900"
  style="border:0;"
  loading="lazy"
  title="OSI Model Brick Sorter"
></iframe>
```
